# DL Projet — Refonte individuelle

Refonte individuelle d'un projet Deep Learning regroupant quatre travaux pratiques.

## Structure

| Dossier | Sujet |
|---|---|
| `tp1_fashionmnist/` | Classification d'images Fashion-MNIST |
| `tp2_consommation_electrique/` | Prévision de la consommation électrique |
| `tp3_regression_vehicules/` | Régression sur données de véhicules |
| `tp4_emotions_transformers/` | Détection d'émotions avec des Transformers |

## Installation

```bash
python -m venv .venv
source .venv/bin/activate  # Windows : .venv\Scripts\activate
pip install -r requirements.txt
```
